# What is FastAPI?

## Definition

FastAPI is a modern, high-performance Python web framework for building RESTful APIs.

It is designed around:

- Python Type Hints
- Automatic Data Validation
- Automatic API Documentation

---

## Built On

FastAPI is built on top of two core libraries:

| Library | Responsibility |
|----------|----------------|
| **Starlette** | Web framework (Routing, ASGI, Middleware) |
| **Pydantic** | Data validation and serialization |

---

## Why FastAPI?

FastAPI was created to solve common problems in API development:

- Reduce boilerplate code
- Improve developer productivity
- Provide automatic validation
- Generate interactive API documentation
- Support asynchronous programming

---

## Key Features

- High Performance
- Automatic Request Validation
- Automatic Response Serialization
- Interactive API Documentation (Swagger & ReDoc)
- Native Async Support
- Type Hint Integration

---

## FastAPI vs Traditional Frameworks

Traditional frameworks often require:

- Manual validation
- Separate documentation
- Additional serialization libraries

FastAPI provides these features out of the box.

---

## Typical Use Cases

- REST APIs
- Backend services
- Microservices
- AI / Machine Learning APIs
- High-performance web services

---

## Summary

- FastAPI is an API-first framework.
- It combines Starlette and Pydantic.
- It emphasizes speed, simplicity, and developer experience.
