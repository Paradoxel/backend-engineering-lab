# Getting Started

## What is FastAPI?

FastAPI is a modern Python web framework for building **high-performance REST APIs**.

It is designed around:

- Python Type Hints
- Automatic Data Validation
- Automatic API Documentation

---

## Built With

- **Starlette** → Web Framework (Routing, ASGI, Middleware)
- **Pydantic** → Data Validation & Serialization

---

## Main Features

- High Performance
- Automatic Validation
- Automatic OpenAPI / Swagger Docs
- Type Hint Support
- Async / Await Support
- Excellent Developer Experience

---

## Django Comparison

| Django | FastAPI |
|---------|----------|
| Full-stack framework | API-first framework |
| Django ORM | Any ORM (SQLAlchemy, Tortoise, etc.) |
| Templates + APIs | APIs primarily |
| Batteries included | Minimal & flexible |

---



## Summary

- FastAPI is an API-first Python framework.
- It combines **Starlette** and **Pydantic**.
- It emphasizes performance, type safety, and developer productivity.
